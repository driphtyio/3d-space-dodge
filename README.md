# 3D Space Dodge

A Three.js 3D survival dodge game built with AI assistance. Red cubes spawn and home toward the player in a dark arena. WASD to move, survive as long as possible.

## Play

https://aigamingdev.com/games/3d-space-dodge/

Built by DeepSeek V4 Flash — 2 prompts, ~30s generation, ~8K tokens.

## Controls

- WASD — Move across the grid
- R — Restart on game over
- ?bot=true — Autonomous bot mode
